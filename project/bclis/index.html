Framework for Broad Class Incremental Learning System

Jan 20, 2021 Project

Duration: October 2020 - January 2021

Mentor: Prof. Krishna K. Sharma, Dept. of Computer Science & Informatics, University of Kota

Details

  • Developed a Novel Learning Framework Broad Class Incremental Learning System (BCILS) to tackle the forgetting problem of Class Incremental Learning.
  • BCILS updates the model when there is training data from unknown classes using the deduced iterative formula.
  • Proposed approach tested on sensor-induced physiological signals.
